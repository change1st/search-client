# ES/OpenSearch查询工具

这是一个基于Electron和React开发的ES/OpenSearch查询工具，允许用户轻松连接到ES或OpenSearch服务器并执行各种查询操作。

## 功能特点

- 连接到ES/OpenSearch服务器
- 保存和管理多个连接配置
- 查看服务器信息（类型、版本）
- 查看可用的索引列表
- 执行高级查询（DSL查询）
- 格式化和压缩JSON查询
- 查询历史记录管理
- 键盘快捷键支持

## 开始使用

### 安装依赖

```bash
npm install
```

### 运行应用

```bash
npm start
# 或者使用electron直接运行
npx electron .
```

## 如何使用

### 1. 连接到ES/OpenSearch服务器

1. 在连接面板中输入以下信息：
   - **连接URL**：ES/OpenSearch服务器的URL，例如 `http://localhost:9200` 或 `https://your-es-server.com:9200`
   - **用户名**：如果服务器需要认证，请输入用户名
   - **密码**：如果服务器需要认证，请输入密码

2. 点击**测试连接**按钮验证连接是否成功

3. 点击**连接**按钮建立连接

4. 连接成功后，应用将显示服务器信息和索引列表

### 2. 执行查询

1. 从索引下拉列表中选择要查询的索引，或选择**全部索引**

2. 在查询编辑器中输入JSON格式的查询DSL，例如：

   ```json
   {
     "query": {
       "match_all": {}
     },
     "size": 10
   }
   ```

3. 使用工具栏按钮格式化或压缩JSON

4. 点击**执行查询**按钮运行查询

5. 查询结果将显示在结果面板中

### 3. 保存连接配置

1. 连接成功后，输入连接名称
2. 点击**保存连接**按钮
3. 保存的连接可以在**已保存连接**列表中查看和管理

## 支持的查询类型

### 高级查询（DSL）
支持完整的ES/OpenSearch查询DSL语法，包括：
- match、term、range等基本查询
- bool查询（must、should、must_not、filter）
- 聚合查询
- 排序、分页等

### 基础查询
简单的过滤查询界面（开发中）

### 聚合查询
专门的聚合查询界面（开发中）

## 键盘快捷键

- **Ctrl/Cmd + K**：聚焦查询编辑器
- **Ctrl/Cmd + L**：清除查询
- **Ctrl/Cmd + Enter**：执行查询

## 常见问题

### 连接失败
- 检查URL是否正确
- 验证用户名和密码是否有效
- 确认服务器是否允许外部连接
- 检查网络连接和防火墙设置

### 查询错误
- 确保JSON格式正确
- 检查查询语法是否符合ES/OpenSearch版本要求
- 确认索引是否存在且具有适当的权限

## 技术栈

- Electron：跨平台桌面应用框架
- React：前端UI库
- ES/OpenSearch API：与服务器通信

## 注意事项

- 请谨慎处理敏感信息，如密码等
- 大型查询可能会消耗大量资源，请适当限制查询范围
- 建议在开发环境中先测试查询，再在生产环境中执行
